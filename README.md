# ele-vue3-ts
## 1. 项目介绍

此项目作为模板使用

### 1.1 技术栈

`Vue3`、`vue-router`、`pinia`、`TypeScript`、`Element-Plus`

- 使用setup组合式开发
- elementPlus组件库(支持组件按需导入和图标自动导入)



## 2. 项目命令
```
npm install
```

### 开发环境编译运行
```
npm run serve
```

### 生产环境编译运行
```
npm run build
```

### 修复eslint代码风格
```
npm run lint
```
